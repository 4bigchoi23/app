# app/masonry-ui

부트스트랩 5 버전에서 `.card-column` 클래스가 [드랍](https://getbootstrap.com/docs/5.3/migration/#card)되었다.  
요소의 용적을 구한 다음 `absolute`로 `position`을 재할당하므로, 이미지 등으로 인하여 페이지 로딩이 느린 경우 `imagesLoaded` 플러그인이 필요할 수 있다.
